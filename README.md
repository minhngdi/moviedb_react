## Movie Database React App - 映画のデータベースアプリ

To run app:

1.   #### `yarn install`
2.   #### `yarn start` 

Instruction:

1. In search bar, type movie's name you want to search (Ex: Flash)
2. Press Enter
3. Click at the movie you want to know more about it
